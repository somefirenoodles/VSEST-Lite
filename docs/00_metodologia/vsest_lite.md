# VSEST-Lite aplicado a PSIREG

## Definición

VSEST-Lite es una adaptación documental de VSEST 29110. Su objetivo es registrar, controlar y verificar productos de trabajo del proyecto sin construir una plataforma adicional.

## Justificación

VSEST 29110 propone almacenar productos de trabajo, registrar estados, facilitar trazabilidad, visualizar avance y evaluar cumplimiento documental contra ISO/IEC 29110. Para PSIREG, estos principios se implementan con GitHub: archivos Markdown, historial de commits, issues, evidencias y registros de revisión.

## Alcance de la adaptación

| Elemento VSEST 29110 | Adaptación VSEST-Lite |
|---|---|
| Producto de trabajo | Archivo Markdown, evidencia o reporte. |
| Estado del producto | Campo obligatorio en cada artefacto. |
| Repositorio de productos | Repositorio GitHub. |
| Trazabilidad | Matriz de ejecución en `docs/02_trazabilidad/`. |
| Progreso | Issues, commits y checklist documental. |
| Retroalimentación | Observaciones del equipo y resultados de revisión. |

## Regla de uso

Cada artefacto debe incluir código, fase, estado, responsable, evidencia, observaciones y resultado de revisión.
