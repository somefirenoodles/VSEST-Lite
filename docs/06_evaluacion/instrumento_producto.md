# Instrumento de evaluación del producto

## Escala

| Valor | Resultado | Criterio |
|---|---|---|
| 2 | Cumple | El criterio está implementado y existe evidencia verificable. |
| 1 | Cumple parcialmente | El criterio existe, pero presenta evidencia incompleta u observaciones. |
| 0 | No cumple | El criterio no existe, falla o no tiene evidencia. |

## Instrumento

| ID | Característica | Criterio | Evidencia | Resultado | Observación |
|---|---|---|---|---|---|
| EV-001 | Adecuación funcional | El sistema permite completar los flujos principales definidos. | Pendiente | 0 | Pendiente. |
| EV-002 | Seguridad | Las funciones críticas requieren autenticación y autorización. | Pendiente | 0 | Pendiente. |
| EV-003 | Usabilidad | Los formularios presentan mensajes claros y comprensibles. | Pendiente | 0 | Pendiente. |
| EV-004 | Fiabilidad | Las operaciones principales producen resultados esperados. | Pendiente | 0 | Pendiente. |
| EV-005 | Mantenibilidad | Los documentos y evidencias permiten localizar cambios por módulo. | Pendiente | 0 | Pendiente. |

## Regla de decisión

Si hay duda entre dos valores, se asigna el menor hasta que exista evidencia suficiente.
