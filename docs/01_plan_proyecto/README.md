# Plan de proyecto

## Alcance

El trabajo aplica VSEST-Lite a PSIREG para controlar la línea base, el proceso, el diagnóstico, el Plan SQA, el instrumento del producto, los riesgos y el cierre metodológico-documental.

## Responsabilidades por rol

| Rol | Responsabilidad |
|---|---|
| Coordinador del proceso | Mantener planificación, estados, riesgos y cierre. |
| Responsable de requisitos | Verificar RF, RNF, casos de uso y trazabilidad. |
| Responsable del artefacto | Elaborar el contenido, atender observaciones y adjuntar evidencia. |
| Responsable de pruebas | Preparar casos, datos y resultados cuando exista producto evaluable. |
| Revisor de calidad | Aplicar revisiones y auditorías sin aprobar evidencia incompleta. |
| Responsable técnico | Preparar entorno, herramientas y mediciones. |

## Cierre

Un artefacto solo pasa a Cerrado cuando posee evidencia verificable y no conserva observaciones abiertas. El cierre del proceso documental no equivale a funcionamiento de PSIREG.
