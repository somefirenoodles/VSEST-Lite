# Identificacion del proyecto PSIREG

PSIREG es el proyecto referencia escogido para realizar la aplicación de aseguramiento del plan de calidad, al analizar el contexto del proyecto, concluimos con que se utilizarían las normas ISO 29110 en conjunto con una metodología agil (Kanban) para realizar las tareas definidas por el estandar. En conjunto, tambien se hace uso de ISO 25000 en la etapa de calidad de producto, permitiendo una mayor facilidad en la formalización de criterios de evaluación sobre producto.
Muñoz et al. plantea el sistema VSEST 29110 Tool como herramienta para el seguimiento, registro, asignación y automatización de tareas para la implementación del ISO 29110 en su perfil básico. VSEST-Lite nace como implementación técnica para la colaboración en el proyecto PSIREG y otros con frameworks basados en metodologías ágiles. Permitiendo al equipo, elaborar artefactos, registrar tareas, controlar el estado de avances, vincular evidencias y generar matrices que apoyen el cumplimiento del plan de calidad propuesto de PSIREG.
