# Identificacion del proyecto PSIREG

PSIREG es una aplicación web institucional propuesta para la Dirección Nacional de Orientación Psicológica de la Universidad Tecnológica de Panamá. Su línea base procede de la Visión y la Especificación de Requerimientos de Software. El problema central es la gestión manual y dispersa de expedientes, citas, actividades, documentos y reportes, condición que afecta la disponibilidad de la información y aumenta el riesgo sobre datos psicológicos sensibles.

El producto se posiciona como una solución institucional autónoma que centraliza esa información y aplica acceso por perfil. La línea base vigente contiene 19 requerimientos funcionales, 22 requerimientos no funcionales y 19 casos de uso. VSEST-Lite se utiliza para controlar el trabajo y la evidencia; no se presenta como evidencia de implementación de PSIREG.
