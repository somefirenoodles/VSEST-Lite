# Proyecto de referencia: PSIREG

## Identificacion del proyecto

El proyecto seleccionado como referencia para la aplicacion de VSEST-Lite es PSIREG, un sistema academico-institucional orientado a la gestion de servicios de orientacion psicologica. El sistema integra procesos relacionados con expedientes psicologicos, citas, documentos, actividades e impresion de informacion institucional.

## Justificacion de seleccion

PSIREG es adecuado para aplicar un framework de calidad porque maneja informacion sensible y requiere control sobre trazabilidad, acceso, seguridad, evidencia verificable y consistencia documental. El proyecto cuenta con documentacion previa de arquitectura, casos de uso, matriz de trazabilidad, prototipos de pantalla y vistas arquitectonicas, lo cual permite evaluar tanto el proceso como el producto desde una perspectiva de aseguramiento de calidad.

## Problema identificado

El problema principal que atiende PSIREG es la gestion manual o dispersa de expedientes, documentos y procesos de atencion psicologica. Esta situacion puede generar demoras en la consulta de informacion, dificultad para generar reportes, dependencia de registros fisicos y menor control sobre datos institucionales sensibles.

## Posicionamiento del producto

PSIREG se posiciona como una solucion de apoyo administrativo y operativo para la Direccion Nacional de Orientacion Psicologica. Su finalidad es centralizar la informacion, digitalizar documentos y expedientes, gestionar citas y actividades, y facilitar el acceso controlado a datos relevantes para usuarios autorizados.

## Alcance inicial para VSEST-Lite

Para la primera fase de uso de VSEST-Lite, el proyecto se registra como artefacto de referencia. Esta fase permite documentar el punto de partida, asociar evidencias existentes y preparar la trazabilidad hacia actividades posteriores como definicion del proceso de desarrollo, diagnostico, propuesta de mejora, plan SQA e instrumento de evaluacion del producto.

## Evidencias base

| Evidencia | Uso dentro de VSEST-Lite |
