# VSEST-Lite

Repositorio documental para aplicar **VSEST-Lite** como soporte de calidad del proyecto PSIREG.

VSEST-Lite integra ISO/IEC 29110, Kanban y el control de evidencia necesario para el Plan SQA y el instrumento basado en SQuaRE. La herramienta demuestra control documental y del proceso; no demuestra que PSIREG se encuentre implementado o que sus características de calidad hayan sido verificadas.

## Regla operativa

Cada artefacto conserva responsable, fase, estado, evidencia, resultado, observación y próxima acción. Los criterios del producto utilizan uno de estos resultados: **Cumple**, **Cumple parcialmente**, **No cumple**, **No evaluado** o **No ejecutable**.
